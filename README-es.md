# ☁️ Recursos — AWS para Desarrolladores
 
Un repositorio dedicado a mis estudios sobre **Amazon Web Services (AWS)**, cubriendo desde los fundamentos de la computación en la nube hasta servicios más avanzados de computación, almacenamiento, bases de datos y seguridad.
 
---
 
## 📚 Contenido de los estudios:
 
### Nivel 1 — Fundamentos de Cloud y AWS
- Modelos de servicio: IaaS, PaaS y SaaS
- Arquitectura global de AWS: regiones y zonas de disponibilidad
- Crear y configurar tu cuenta
- Navegación por la consola de AWS
### Nivel 2 — Computación y Almacenamiento
- Servicio Amazon EC2 (Elastic Compute Cloud)
- Amazon S3 (Simple Storage Service)
- Configuración de AWS CLI
- Gestión de recursos vía línea de comandos (AWS CLI)
### Nivel 3 — Bases de Datos
- Servicios de bases de datos gestionadas en AWS
- Diferencias entre bases de datos relacionales y no relacionales en la nube
- Configuración y conexión a instancias de bases de datos
### Nivel 4 — Seguridad y Acceso
- AWS IAM (Identity and Access Management)
- Control de acceso, usuarios, grupos y políticas
- Buenas prácticas de seguridad en la nube

## 🚀 ¿Por qué AWS?
 
AWS es la plataforma cloud líder en el mundo, utilizada por millones de empresas para desarrollar, implementar y escalar aplicaciones. Dominar sus servicios es una habilidad esencial para cualquier desarrollador que trabaja con infraestructura moderna.
