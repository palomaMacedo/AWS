## ⚖️ Load Balancer

## 🏨 It's the "hotel receptionist"

Receives everyone at the door
Directs each person to the right queue
Prevents a single cashier from getting overloaded



## 🏥 Health Check

##  🩺 It's the "medical check-up"

Doctor = Load Balancer
Check-up = Health Check
Patient = EC2 Server


## 🖥️ Cluster

##  ⚽ It's the "soccer team"

Player = Server
Entire team = Cluster
Everyone works together toward the same goal



##  🖼️ AMI

## 🎂 It's the "cake mold"

The mold = AMI
The cake = EC2 Instance
With the same mold you can make multiple identical cakes!



##  🌍 Regions and AZs

# 🏬 It's the "mall and its stores"

Mall = Region (ex: São Paulo)
Stores = AZs (data centers inside the region)
If one store closes, the mall stays open!



##  🔐 IAM

##  🪪 It's the "employee badge"

Company = AWS
Employee = IAM User
Badge = Permissions and policies
Each person accesses only what they need!



## ⚡ Lambda vs EC2

## 👔 EC2 is the "full-time employee"

You pay the salary every month
Even if they sit idle doing nothing



## 💻 Lambda is the "freelancer"

You pay only when they work
If nothing runs, you pay nothing!



## 🗄️ S3

☁️ It's the "AWS Google Drive"

Stores any type of file
Accessible from anywhere
Practically unlimited

## ⚽ It's the "coach and the team"
