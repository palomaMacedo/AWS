# ☁️ Resources — AWS for Developers
 
A repository dedicated to my studies on **Amazon Web Services (AWS)**, covering everything from the fundamentals of cloud computing to more advanced computing, storage, database, and security services.
 
---
 
## 📚 Study content:
 
### Level 1 — Cloud and AWS Fundamentals
- Service models: IaaS, PaaS, and SaaS
- AWS Global Architecture: Regions and Availability Zones
- Create and Set Up Your Account
- Navigating the AWS Management Console
### Level 2 — Compute and Storage
- Amazon EC2 (Elastic Compute Cloud) Service
- Amazon S3 (Simple Storage Service)
- Configuring the AWS CLI
- Managing resources via the Command Line Interface (AWS CLI)
### Level 3 — Databases
- Managed Database Services on AWS
- Differences Between Relational and Non-Relational Databases in the Cloud
- Configuring and Connecting to Database Instances
### Level 4 — Security and Access
- AWS IAM (Identity and Access Management)
- Access Control, Users, Groups, and Policies
- Cloud Security Best Practices
