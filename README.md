# ☁️ Estudos — AWS para Desenvolvedores
 
Repositório dedicado aos meus estudos sobre **Amazon Web Services (AWS)**, cobrindo desde os fundamentos de Cloud Computing até serviços mais avançados de computação, armazenamento, bancos de dados e segurança.
 
---
 
## 📚 Conteúdo dos Estudos
 
### Nível 1 — Fundamentos de Cloud e AWS
- Modelos de serviço: IaaS, PaaS e SaaS
- Arquitetura global da AWS: regiões e zonas de disponibilidade
- Criação e configuração de conta gratuita
- Navegação pelo console AWS
### Nível 2 — Computação e Armazenamento
- Serviço EC2 (Elastic Compute Cloud)
- Armazenamento com S3 (Simple Storage Service)
- Configuração da AWS CLI
- Gerenciamento de recursos via linha de comando
### Nível 3 — Bancos de Dados
- Serviços gerenciados de banco de dados na AWS
- Diferenças entre bancos relacionais e não-relacionais na nuvem
- Configuração e conexão com instâncias de banco de dados
### Nível 4 — Segurança e Acesso
- IAM (Identity and Access Management)
- Controle de acesso, usuários, grupos e políticas
- Boas práticas de segurança na nuvem

## 🚀 Por que AWS?
 
A AWS é a plataforma de cloud líder mundial, usada por milhões de empresas para desenvolver, implantar e escalar aplicações. Dominar seus serviços é uma habilidade essencial para qualquer desenvolvedor que trabalha com infraestrutura moderna.
